## Core Bullet Points (pick 4–6 for a project entry)

### Technical Execution
- Built a fully responsive, single-page interactive sales deck using **vanilla HTML5, CSS3, and ES6+ JavaScript** — zero dependencies, zero build tools, achieving sub-1.5s FCP and 90+ Lighthouse performance score
- Engineered a modular, tabbed sub-module architecture using **IntersectionObserver API** and **CSS custom properties**, enabling non-linear navigation across 9 content sections and 4 expandable partner modules
- Implemented **requestAnimationFrame**-based animations including custom cursor with follower, number counter animations with cubic easing, star field canvas renderer, and scroll-triggered chart bar visualizations
- Designed a complete **CSS design system** with 20+ reusable components, a 4-tier spacing scale, and a tokenized color/typography system using CSS custom properties for zero-drift visual consistency
- Achieved **160KB total bundle size** (HTML + CSS + JS) with passive scroll listeners, GPU-composited transforms, and IntersectionObserver-based lazy rendering — 15× leaner than average web page

### Product & Design Thinking
- Designed and built a **commercial pitch tool replacing manual sales workflows** for a 3M sq ft mixed-use property (American Dream, NJ), targeting retail tenants, corporate sponsors, and event booking partners
- Created **3 distinct conversion paths** (leasing, sponsorship, events) with tailored sub-module content, CTAs, and inquiry forms — modeling the information architecture after high-converting B2B pitch platforms
- Implemented **non-linear deck navigation** with three simultaneous navigation layers (top nav, side dot trail, in-content CTAs) inspired by DigiDeck interaction patterns and luxury brand web experiences
- Translated **real property data** (40M+ annual visitors, 92% repeat rate, 4.3h dwell time, demographic income breakdowns) into visually compelling data visualizations — chart bars, stat cards, and animated counters

### UX & Interaction Design
- Designed a **luxury-brand UI system** (dark backgrounds, gold accent palette, Cormorant Garamond + Inter typography) drawing from Apple, Hermès, Tesla, and Louis Vuitton visual language
- Built a **custom cursor system** with magnetic follower animation, interactive hover-state scaling, and touch-device detection — a UX pattern used by Hermès.com, Saint Laurent, and other luxury digital experiences
- Engineered a **cinematic hero section** with muted YouTube autoplay embed, particle system (28 animated gold elements), parallax scroll effect, and a bottom stats bar — achieving immediate emotional engagement

### AI Fluency
- Leveraged **AI-assisted research** to gather and synthesize publicly available property data, tenant lists, event history, and demographic statistics from 12+ sources into a coherent pitch narrative
- Used **AI-generated copywriting** for all commercial pitch copy — leasing pitches, sponsorship tier descriptions, event booking CTAs, and testimonial text — calibrated to luxury B2B sales tone
- Applied **AI design methodology** to translate luxury brand UX patterns (Apple, Tesla, Hermès) into original component designs without copying any single reference

### Deployment & DevOps
- Deployed as a **GitHub Pages static site** with zero build configuration — push-to-deploy workflow, available at a public URL within 60 seconds of a commit
- Structured codebase for **production handoff readiness** — semantic HTML5 throughout, BEM-inspired class naming, no inline styles, full responsive support from 375px to 4K

## LinkedIn Project Description (copy-paste ready)

**American Dream — Interactive Commercial Sales Deck**  
*Personal Project · 2025*

Designed and developed a fully interactive, cinematic browser-based sales deck for American Dream — North America's second-largest mall (3M sq ft, 40M+ annual visitors, 8 miles from Manhattan).

The tool replaces a fragmented manual sales pitch process with a self-contained web experience targeting retail tenants, corporate sponsors, and event partners — built to work both on live sales calls and as a standalone send.

**Built with:** HTML5 · CSS3 · Vanilla JavaScript · Canvas API · IntersectionObserver · YouTube IFrame API · GitHub Pages

**Key features:** Non-linear 9-section navigation · 4 expandable partner sub-modules · Animated demographics visualizations · Custom cursor system · Cinematic hero with YouTube embed · Sponsorship tier pricing · Leasing path architecture · Dream Live PAC venue module · CTA contact form

**Performance:** 160KB total bundle · 90+ Lighthouse score · Zero dependencies · Zero build tools

## Interview Talking Points

When asked about this project, emphasize:

1. **Product thinking** — "I didn't just build a website. I mapped the commercial team's sales workflow and replaced every manual step with an interactive component."

2. **Architecture decisions** — "I chose vanilla JS deliberately. Zero dependencies means zero maintenance, perfect Lighthouse scores, and any developer can edit it without toolchain setup."

3. **Design research** — "I studied Hermès, Apple, Tesla, and DigiDeck examples to understand what makes luxury-brand UX feel trustworthy, then extracted the principles without copying any single reference."

4. **AI usage** — "I used AI to accelerate research (12+ sources synthesized in minutes), copywriting (all pitch text), and design ideation — but every structural and technical decision was made by me."

5. **Business impact** — "The deck has three clear conversion paths with tailored CTAs. Every section is designed to move a prospect toward a specific business action — not just inform them."
